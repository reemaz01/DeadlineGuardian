# 🚀 Deadline Guardian AI

An AI-powered productivity and deadline management platform that helps students and professionals organize tasks, reduce procrastination, and meet deadlines with intelligent planning and real-time insights.

🌐 **Live Demo:** https://deadline-guardian-ai-1081861451627.asia-southeast1.run.app

---

## 📖 Overview

Deadline Guardian AI is a smart productivity assistant designed to transform the way users manage deadlines. Instead of simply storing tasks, the application analyzes deadlines, predicts risks, detects procrastination, and generates AI-powered execution plans to maximize productivity.

Whether you're a student managing assignments or a professional juggling multiple projects, Deadline Guardian AI helps you stay ahead of your schedule with personalized recommendations and intelligent task management.

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/yourusername/deadline-guardian-ai.git
```

## Navigate into Project

```bash
cd deadline-guardian-ai
```

## Install Dependencies

```bash
npm install
```

## Start Frontend

```bash
npm run dev
```

## Start Backend

```bash
node server.js
```
